here we are going with the id selector

ID selector uses the ID attribute of an HTML element to select one unique element on a page

To use an ID selector in CSS, you simply write a hashtag (#) followed by the ID of the element. Then put the style properties you want to apply to the element in brackets.

IDs should be unique on a page

you won't be able to reuse it within the same webpage

the id selectors are not part of the DOM
